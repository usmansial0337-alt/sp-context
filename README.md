# 🧠 sp-context - Give AI agents your team's knowledge

[![](https://img.shields.io/badge/Download-sp--context-blue)](https://github.com/usmansial0337-alt/sp-context)

## What is sp-context?

Teams often struggle to keep their AI assistants informed about internal projects. Most solutions require complex setups, expensive databases, or constant updates. sp-context changes this. It provides your AI agents with a persistent memory of your team's knowledge without the heavy lifting.

This tool uses simple git-based storage and standard search methods to retrieve information. It keeps your data organized and ready for use by your AI tools. It focus on efficiency, using only a small set of data tokens per session to keep costs low and performance high.

## Why use sp-context?

You do not need to manage complicated database systems. sp-context relies on the files you already have. Because it integrates with version control, you keep a history of your team knowledge. 

- Simple setup for any computer user.
- No complex database management.
- Low token usage saves money.
- Reliable search results using standard methods.
- Works with your existing project files.

## 🚀 Getting Started

Once you have installed the program, follow these steps to configure your team memory.

1. Open the sp-context application from your Start menu.
2. Select the folder on your computer that contains the documents or code you want your AI to know.
3. Click the Index button. The application will scan your files and prepare them for search.
4. Once the process bar reaches 100%, your team knowledge is ready.
5. Provide the local address shown in the status bar to your AI agent configuration.

## 🏠 Managing Your Knowledge

You can update your memory at any time. If you add new files to your chosen folder, return to the application and click the Refresh button. This updates the index so your AI agent always has the latest information.

If you decide to stop sharing a specific folder, use the Remove button in the folder list. This clears the memory index and stops the application from scanning those files.

## 🛠️ Troubleshooting

If you encounter issues, check these common fixes.

- Path issues: Ensure the folder you select does not have restricted access permissions.
- File types: The tool currently prioritizes text-based files, code files, and standard documentation formats. Ensure your documents are in these formats.
- Performance: If the application seems slow, clear your cache from the settings menu.

## Use Cases

### For Development Teams
Developers often share code patterns and internal documentation. sp-context allows AI assistants to scan these files instantly. This prevents the need to paste large chunks of code into an AI chat window constantly.

### For Project Management
Keep lists of project requirements, meeting notes, and status updates in a central folder. The AI scans these files to answer questions about project history without needing you to search through emails or messaging apps.

### For Knowledge Sharing

## Security and Privacy

Your data stays on your machine. We do not store your files on external servers. The software processes your documents locally and only shares relevant snippets when you ask a specific question. This ensures that your private company information remains under your control.

## Updates and Maintenance

The application checks for updates automatically when you open it. If a new version is available, you will see a prompt to download the latest files. We recommend updating regularly to ensure you have the best search speed and reliability.
